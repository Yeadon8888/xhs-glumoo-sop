# Glumoo 宠物内容 SOP（马来西亚中高端版）

## 固定目标
用于每天稳定产出 Glumoo 小红书内容：
- 文案
- 5 张配图
- 审稿版

并确保以下元素统一：
- SKU
- 宠物品种
- 宠物外貌锚点
- 包装
- 图中文字
- 场景层级
- 人设气质

## SKU 轮换
- SKU1 = 多维力
- SKU2 = 跃力
- SKU3 = 畅清

默认按日轮换，不长期固定单一 SKU。

## 宠物品种轮换池
### 狗
1. 玩具体贵宾（泰迪风）
2. 博美
3. 比熊
4. 金毛
5. 雪纳瑞

### 猫
1. 英短
2. 布偶
3. 波斯
4. 缅因
5. 曼赤肯

## 宠物一致性规则
- 同一篇内容必须固定一个宠物品种。
- 同一组图必须是同一只宠物。
- 图片1定义外貌锚点，后续图片逐字复用。
- 若是玩具体贵宾（泰迪风），禁止画成标准赛级长嘴贵宾、比熊或其他卷毛犬。

## 文案与图片统一规则
- 文案里写什么宠物品种，图里就必须是什么品种。
- 文案与配图都必须服务同一 SKU。
- 每张图要有自己的短文案，不允许只出纯场景图。
- 封面必须有与标题强关联的短文案。

## 包装规则
- SKU1 只能使用 `sku1.png` + `sku1小包装.png`
- SKU2 只能使用 `sku2.jpg` + `sku2小包装.png`
- SKU3 只能使用 `sku3.jpg` + `sku3小包装.png`
- 不允许串包装。
- 凡出现拌饭、倒粉、骗水、冲泡场景，必须使用当前 SKU 正确小包装。

## 马来西亚中高端场景规则
### 场景定位
整体必须服务：
- 马来西亚中高端华人家庭
- 精致养宠
- 家居整洁、有秩序、有审美
- 低调有钱，不浮夸

### 场景映射
- 城市场景 → 高端社区 / 高档 condo / 高级公寓 / 城市高端住宅
- 低密度场景 → 高端别墅 / landed villa / 花园庭院住宅
- 半户外 → 车棚、庭院、泳池边、前院草坪、露台、阳台
- 室内 → 大客厅、餐厨岛台、落地窗餐厅、主卧角落、衣帽区、楼梯口

### 马来西亚识别元素
每张图至少有 1-2 个：
- high-end condo
- landed villa
- terrace house
- 铁花窗 / grille
- 吊扇
- 复古花砖 / 水磨石地
- 藤编家具
- 车棚走廊
- 热带植物
- 强热带日照与阴影

## 主人同框规则
- 默认优先主人同框。
- 优先画法：
  - 主人拌饭
  - 主人喂食
  - 主人散步
  - 主人陪伴互动
  - 主人坐沙发，宠物自然靠近
- 不要永远拍成宠物单独摆拍。

## 豪车规则
- 可以低调露出高端 SUV / sedan。
- 只能做背景身份线索。
- 不可喧宾夺主。
- 不可拍成车广告。
- 适合出现在：
  - 车棚
  - driveway
  - condo 地库
  - 门前接送场景

## search / discovery 规则
- search：解决问题、判断方案、真实使用、长期管理
- discovery：向往感、生活方式、轻决策、收藏感

## 输出要求
每次必须产出：
1. 标题
2. 正文
3. 图片1-5提示词
4. 每张图的图中文字
5. 每张图的人物安排
6. 每张图的场景层级
7. 每张图的豪车线索（如适合）
8. 审稿版

## 当前默认日更原则
以后每天按以下维度轮换：
- SKU 轮换
- 宠物品种轮换
- 场景轮换
- 室内 / 户外轮换
- 内容类型轮换

但必须始终保持：
- 中高端马来西亚华人养宠人设
- 文案 / 图 / 包装 / 宠物一致
